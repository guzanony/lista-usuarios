"Observações:"
O BACKEND FOI FEITO PELO INTELLIJ E O FRONTEND PELO VSCODE. PELO INTELLIJ ESTOU UTILIZANDO O MAVEN. CASO NÃO SEJA RECONHECIDO, BASTA CLICAR COM O BOTÃO DIREITO NO 'pom.xml' E CLICAR EM 'add maven project'. NO H2, A JDBC URL É 'jdbc:h2:file:./data/usuarios'. FOI FEITO ESTE PROJETO EM ANGULAR. O ARQUIVO INDEX.HTML COM AS INFORMAÇÕES ESTARÁ TAMBEM NO BACKEND SPRING BOOT MAS TAMBÉM ESTÁ NO FRONTEND ANGULAR PARA FAZER OS DIRECIONAMENTOS DAS TELAS. POR PADRÃO QUANDO RODAR O PROJETO, VAI SER ABERTO ESTÁ TELA. PARA RODAR O PROJETO ANGULAR, BASTA ABRIR O PROJETO PELO VSCODE, ABRIR O TERMINAL E EXECUTAR O COMANDO 'npm install' E EM SEGUIDA 'npm run start' (necessário ter a versão do node >= 18). PARA ACESSAR NO NAVEGADOR, VAI SER STARTADO NA PORTA 'localhost:4200'.
